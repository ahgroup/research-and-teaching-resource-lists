Resources to learn R

List maintained by Andreas Handel.

If you know of other good resources, please contribute or let me know.

Last updated 12/29/2016.

Lists by others

  • [A list of the best R tools] (https://github.com/qinwf/awesome-R)

Online Courses for R

  • DataCamp R online tutorials - All levels of difficulty, good interactive exercises with a nice graphical interface. Some courses are completely free, for others a paid subscription is required.

  • Swirl - An R package that teaches R interactively inside R. Not as polished as DataCamp, but good interactivity, great motivating feedback/messages, beginner and intermediate levels. Completely free.

  • Try R, a free course at codeschool.com

  • Someone else’s list of R resources

R books and tutorials

  • YaRI - a short R tutorial I wrote (before many of the newer & better resources like swirl and DataCamp existed)

  • YaRrr! The Pirate’s Guide to R: A nice R introduction with a piratey bend

  • Computerworld’s R beginner guide

  • ‘R Meta book’ - a list of resources on specific topics:

  • Springer has a series of books called ‘Use R’ - A long and growing series of books on a lot of different R topics. From beginner to advanced. Many university libraries have a subscription to Springer Online, which allows you to get the books for free.

  • Roger Peng’s “R programming for data science” book on Leanpub. Name your own price.

  • “Advanced R” by Hadley Wickham for more advanced R coding

General R Online Resources

  • Quick R - a great resource for many ‘how do I’ questions of common data analysis and graphics tasks.

  • Cookbook for R - another useful collection of ‘how do I do X’ information.

  • R bloggers an aggregation of blogs about all things R

  • A great collection of R resources and tutorials

  • A very unique Pinterest board :)

  • Convenient reference/cheat sheets for some important R tasks

  • Resource collection to do useful data analysis related tasks in R

  • Collection of links for R learning

  • R Documentation: - a webpage and R package that tries to enhance the help/documentation files for R and its packages.

R packages

  • The Task View sorting R packages into categories is often useful

  • Tutorials for the dplyr package

  • How to quickly write an R package

  • Hadley’s book on R Packages - Free online version of the book. This book and a bit of googling is all you need to successfully make your own R packages.
